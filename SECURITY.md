# Security Policy for JavaTorch

## Supported Versions

Below are the versions of JavaTorch that are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within JavaTorch, please **do not** report it publicly via GitHub issues. 

Instead, please send an email to the project maintainers directly. We take all security vulnerabilities seriously and will work to verify and address them as quickly as possible.

### What to include

- Vulnerability type (e.g., buffer overflow, arbitrary code execution)
- A description of the issue.
- Step-by-step instructions to reproduce the problem.
- Any potentially affected versions.

We will try to address it as soon as possible and aim to release a patch or a workaround before the vulnerability is made public.
